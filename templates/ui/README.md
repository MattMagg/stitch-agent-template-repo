# UI Templates

Reusable UI template snippets and starter layouts.

## Purpose

This directory is a place to collect reusable HTML/CSS/component patterns that you find yourself using across projects. It's not populated by default — add templates as you develop your own patterns.

## Suggested Templates

As you work through projects, consider saving reusable patterns here:

- **Layout shells**: Sidebar + main content, top nav + content, etc.
- **Component patterns**: Card grids, data tables, form layouts, hero sections
- **Page skeletons**: Common page structures with placeholder content

## Format

Use whatever format matches your workflow:

- **HTML files**: For framework-agnostic templates
- **Framework components**: For React, Vue, Svelte, etc.
- **CSS snippets**: For reusable style patterns
