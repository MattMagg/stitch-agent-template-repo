# References

Store reference materials used to inform design decisions here.

## Contents

- `screenshots/` — reference screenshots and inspiration captures
- `notes.md` — written notes about design references and inspiration

## Usage

1. Before creating a canonical screen, collect screenshots of designs you want to reference
2. Save them to `screenshots/` with descriptive filenames
3. Document what you like about each reference in `notes.md`
4. Reference these materials when writing prompts for Stitch

## Naming Convention

Use descriptive names:
- `stripe-dashboard-sidebar.png`
- `linear-issue-detail.png`
- `competitor-landing-hero.png`

Avoid generic names like `ref1.png` or `screenshot.png`.
