# Tools

Automation stubs and helper scripts for the Stitch workflow.

## Contents

- `stitch/` — Stitch-specific automation stubs (sync, extract, export)

## Philosophy

These are starting points, not complete implementations. They're shell scripts for maximum portability. Replace or extend them in your preferred language as needed.

## Available Stubs

| Script | Purpose |
|---|---|
| `stitch/sync-stub.sh` | Sync Stitch artifacts to local `.stitch/designs/` |
| `stitch/extract-contract-stub.sh` | Extract design contract values into `.stitch/DESIGN.md` |
| `stitch/export-stub.sh` | Export design artifacts from Stitch |
